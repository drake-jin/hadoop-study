# 책 소개
 - 2장은 맵리듀스를 소개
 - 3장은 하둡의 파일 시스템 
 - 4장은 하둡 I/O에 대해서
 - 2,3,4 장은 데이터의 부결성, 압축, 직렬화, 파일 기반의 데이터 구조 등이 주요 내용이다.
 - 5,6,7,8장은 맵리듀스의 고급 기능에 대해 배운다.
 - 9,10 장은 하둡 관리자를 위한것이다.
 - 11,12 장은 HDFS와 맵 리듀스 기반의 분석 플랫폼인 피그와 하이브를 설명한다.
 - 13,14,15 는 HBase, Zookeeper, Sqoop
 - 16장은 사례 연구들을 모아 놓았다.
